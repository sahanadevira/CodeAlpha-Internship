# CodeAlpha_python_Chatbot
Basic Chatbot

Project Overview:
This project is a simple rule-based chatbot developed using Python. The chatbot interacts with users through predefined responses based on the input provided.

Features:
1)Greets the user.
2)Responds to common questions such as "How are you?".
3)Handles unknown inputs gracefully.
4)Allows the user to exit the conversation using the command "bye".

Technologies Used:
1.Python 3
2.Conditional Statements (if-elif-else)
3.Functions
4.Loops
5.Input/Output Operations

Project Structure
basic_chatbot.py
README.md

How to Run
Install Python 3.
Download or clone this repository.
Open a terminal in the project folder.
Run the following command:
python basic_chatbot.py

Sample Interaction
You: hello
Chatbot: Hi!
You: how are you
Chatbot: I'm fine, thanks!
You: bye
Chatbot: Goodbye!

Learning Outcomes
Understanding rule-based chatbot logic.
Using functions and loops in Python.
Handling user input and output.
Implementing conditional statements.
