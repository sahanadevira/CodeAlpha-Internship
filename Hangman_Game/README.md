# CodeAlpha_Hangman_game
This is a Hangman game build with python using random, while loop, if-else, string, lists.
Hangman Game in Python
Project Overview
The Hangman Game is a classic word-guessing game developed using Python. In this game, the computer randomly selects a word from a predefined list, and the player attempts to guess the word one letter at a time. For each incorrect guess, the player loses one attempt. The game continues until the player successfully guesses the entire word or runs out of available attempts.
This project demonstrates the practical application of fundamental Python programming concepts such as loops, conditional statements, lists, string manipulation, user input handling, and the use of Python's built-in random module. It provides an engaging way for beginners to understand game logic and interactive programming.

# Features
Randomly selects a word from a predefined collection.
Displays hidden letters using underscores.
Allows users to guess one letter at a time.
Tracks previously guessed letters.
Provides feedback for correct and incorrect guesses.
Limits the number of incorrect attempts.
Displays a victory message when the word is guessed correctly.
Shows the correct word when the player loses.

Technologies Used
Python 3
Random Module
Working Principle
A random word is selected from a list of words.
The word is displayed as underscores representing hidden letters.
The player enters a letter as a guess.
If the guessed letter exists in the word, all occurrences of that letter are revealed.
If the guessed letter does not exist in the word, the number of remaining attempts decreases.

# Dependencies
This project uses Python's built-in random module
No external libraries or packages are required

The process repeats until: 
The player correctly guesses all letters, or
The player exhausts all available attempts.
The final result is displayed accordingly.
